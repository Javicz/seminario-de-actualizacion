📱 WebComponent de Chat

Abstracción de interfaz de chat con métodos para insertar mensajes

🔧 Uso programático

Puedes usar el método addMessage(texto, esPropio, hora) para insertar mensajes desde JavaScript: 

// Obtener referencia al componente
const chat = document.querySelector('chat-component');

// Agregar mensaje propio
chat.addMessage('¡Hola desde JavaScript!', true, '14:30');

// Agregar mensaje de otro
chat.addMessage('¡Hola! ¿Cómo estás?', false, '14:31');

// Limpiar todos los mensajes
chat.clearMessages();

// Obtener historial de mensajes
const historial = chat.getMessages();
console.log(historial);
